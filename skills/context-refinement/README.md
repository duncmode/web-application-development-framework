# Skill: Context Refinement

Evaluates the completeness of a task's context.md and produces a readiness report.

## What It Does
- Reads the task's context.md
- Checks against the context refinement checklist (ai-instructions.md Section 5)
- Produces a readiness report with Strong/Gaps/Recommendation sections
- Offers to help fill in identified gaps
