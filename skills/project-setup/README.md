# Skill: Project Setup

Automates the creation of a new project from the framework scaffold.

## What It Does
- Copies the project scaffold template
- Links the project to the framework
- Initializes git with main and staging branches
- Creates .gitignore

## How to Invoke
```bash
./scripts/setup.sh <project-name> <project-directory>
```
