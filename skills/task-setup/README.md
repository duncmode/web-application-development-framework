# Skill: Task Setup

Creates a new task folder from the task template and adds the task to the roadmap.

## What It Does
- Copies the task template to tasks/[TASK-ID]-[description]/
- Adds the task to roadmap.md in the appropriate section
- Creates the feedback/assets/ directory

Not yet implemented. Currently manual: copy templates/task-template/ to the tasks/ directory and rename.
