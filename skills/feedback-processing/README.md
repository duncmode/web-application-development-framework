# Skill: Feedback Processing

Organizes raw feedback into structured, actionable items.

## What It Does
- Reads raw feedback from the task's feedback/ directory (text, screenshots, etc.)
- Produces a structured round-N.md with numbered items
- Categorizes items by severity: blocker / should-fix / nice-to-have
- Maps feedback to relevant acceptance criteria where possible
